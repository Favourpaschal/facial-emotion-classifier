# Facial Emotion Recognition (CNN)

This project builds a **Convolutional Neural Network (CNN)** to automatically detect human facial emotions from images. It includes complete steps from data preprocessing and visualization to model training and evaluation.

## Key Features

* Image loading, preprocessing & augmentation

* Emotion class visualization

* CNN-based deep learning model

* Train/validation split

* Performance evaluation with accuracy, confusion matrix & classification report

# Emotion Classes

Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise (7 classes)

# Data

https://www.kaggle.com/datasets/fahadullaha/facial-emotion-recognition-dataset

# Applications

Emotion-aware AI, mental health support, HCI systems, smart surveillance
